Definition:
Ein Netzwerk ist die phisikale und logische verbindung von Computersystemen.
Ein einfaches Netzwerk besteht aus zwei Computersystemen. Sie sind über (ein) Kabel miteinander verbunden.

Vorteile Nachteile
Geld sparen Kaufen von Netzwerkkaeln & Datenserver
Sichern und Speichern von Dateien Verwaltung von großen Netzwerke => komplizierter
Komunikation zwischen Benutzer Ausfall von Server => kein Datenzugriff
Dateien zwischen Benutzer Teilen Vieren können durh den Computer verbreitet werden => weniger sicherheit
Firewall gegen Hacker duch den Netz
LAN Computernetzwerk innerhalb eines räumlich begrenzten Bereich.
